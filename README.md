### 🚀 Featured projects

**[LIZA — Visual Novel Game Engine](https://github.com/Arianwait/LIZA---Visual-Novel-Game-Engine)**
Extensible visual novel engine on Java 21 + JavaFX. Stories are scripted in XML; new commands,
effects, UI panels and mini-games plug in via annotations (Reflections) — without touching the core.

**[LIZA Text Editor](https://github.com/Arianwait/LIZA-text-editor)**
Visual node-graph scenario editor for the LIZA engine. Build the story as a scene graph, edit
dialogue/characters/chapters in a GUI, then validate and export to the engine's XML format.

**[pixai-api-java](https://github.com/Arianwait/pixai-api-java)**
Minimal, dependency-light Java 17 client for the PixAI image-generation REST API v2. Fluent
immutable builders, typed parameters, blocking/async generation, LoRA and webhook support —
only OkHttp + org.json at runtime.

**[OracleIntegrationApp](https://github.com/Arianwait/OracleIntegrationApp)**
Config-driven JavaFX desktop client for Oracle DB — dynamic views and stored-procedure forms
built from a JSON config, with Excel export via Apache POI.
